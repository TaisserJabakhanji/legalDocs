# 📜 Legal Docs Platform

منصة احترافية لإنشاء وإدارة المستندات القانونية مع دعم متعدد اللغات.

## ✨ الميزات

- 🌍 دعم كامل للعربية والإنجليزية (RTL/LTR)
- 🌓 ثيم داكن/فاتح مع انتقالات ناعمة
- 💰 نظام أسعار مرن مع تبديل شهري/سنوي
- 🔍 تحسين كامل لـ SEO مع Schema.org
- 📱 تصميم متجاوب 100% لجميع الشاشات
- ♿ إمكانية وصول ممتازة (Accessibility)

## 🛠️ التقنيات المستخدمة

- **Framework**: Next.js 16.2.4 (App Router + Turbopack)
- **Language**: TypeScript
- **Styling**: CSS Modules + CSS Variables
- **State**: React Hooks + Context
- **i18n**: Custom hook with JSON modules
- **Deployment**: Vercel-ready
